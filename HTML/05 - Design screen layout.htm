For this app we will be using containers for the different screen sections.

  1. First up will be a container for the top screen header
  2. From the top toolbar, click on +Insert
  3. Locate and select the Container control under Layout, not horizontal or vertical
  4. Rename this to: conScreenHeader
  5. Let's change some properties, make sure to click on conScreenHeader

    Property Value
    X 0
    Y 0
    Width App.Width
    PaddingRight 0
    PaddingLeft 0
    Height 55
    Fill RGBA(56,96,178,1)

  6. Next up will be a container that will hold our left gallery menu
  7. Make sure to first click on the Main Screen control
  8. Then from the top toolbar, click on +Insert
  9. Locate and select the Horizontal container control
  10. Rename this one to: conLessonMenu
  11. Move this to the bottom of the Tree view, click on the … then Reorder/Send to back
  12. Change the following properties for conLessonMenu

    Property Value
    X conScreenHeader.X
    Y conScreenHeader.Y + conScreenHeader.Height
    Width Hard coding the value to 375
    Height App.Height - Self.Y

  13. Next up will be a container that will hold our Lesson header
  14. Again, make sure you click on the Main Screen control first
  15. Then from the top toolbar, click on +Insert
  16. Locate and select the Container control, not horizontal or vertical
  17. Rename this one to: conLessonHeader
  18. Move this to the bottom of the Tree view, click on the … then Reorder/Send to back
  19. Change the following properties for conLessonHeader

    Property Value
    X conLessonMenu.X + conLessonMenu.Width
    Y conLessonMenu.Y
    Width App.Width - Self.Y
    PaddingRight 15
    PaddingLeft 10
    Height 80

  20. Next up will be a container that will hold the controls for the content
  21. Make sure to click on the Main Screen control first
  22. From the top toolbar, click on +Insert
  23. Locate and select just the Container control, not vertical or horizontal
  24. Rename this one to: conLessonContent
  25. Move this to the bottom of the Tree view, click on the … then Reorder/Send to back
  26. Change the following properties for conLessonContent

    Property Value
    X conLessonHeader.X
    Y conLessonHeader.Y + conLessonHeader.Height
    Width conLessonHeader.Width
    PaddingRight 15
    PaddingLeft 10
    Height App.Height - Self.Y

  27. Next up will be a container that will hold Admin related stuff
  28. Make sure to click on the Main Screen control first
  29. From the top toolbar, click on +Insert
  30. Locate and select the Vertical container control
  31. Rename this one to: conAdminStuff
  32. Move this to the bottom of the Tree view, click on the … then Reorder/Send to back
  33. We don't user borders, so we will use the BorderThickness property to hold its intId
  34. Change the following properties for conAdminStuff

    Property Value
    X conLessonContent.X
    Y conLessonContent.Y
    Width conLessonContent.Width
    Visible galLessonMenu.Selected.intId = Self.BorderThickness

    (Yes this will show an error for now, next step we add the gallery)
    PaddingRight 15
    PaddingLeft 10
    LayoutGap 30
    Height conLessonContent.Height
    BorderThickness 1000 (We get this from the associated intId in colMenuItems)
    BorderStyle BorderStyle.None

  35. Now move/rearrange your containers in the Tree view a bit more logical as follows
    1. conScreenHeader
    2. conLessonMenu
    3. conLessonHeader
    4. conLessonContent
    5. conAdminStuff
  36. Good spot to save your app
