Now that most functionality is present, lets update our screen header.

  1. Select the control conScreenHeader in the Tree view
  2. Then from the top toolbar, click on +Insert
  3. Locate and select the Text label control
  4. Rename this one: lblPctComplete
  5. Now let's do it again, click on +Insert
  6. Select Text label
  7. Rename this one: lblAppTitle
  8. And again, click on +Insert
  9. Select Text label
  10. Rename it: lblMinsRemaining
  11. Now select all 3, on Mac its Shift+Click on each
  12. We can now edit common properties

    Property Value
    X 0
    Y 0
    Width 250
    VerticalAlign VerticalAlign.Middle
    Size 13
    Height Parent.Height
    Color Color.White

  13. Ok, let's work on lblPctComplete, in the Tree view, click it
  14. Update its properties as follows

    Property Value
    X 30
    Y 0
    Text 
    RoundUp( 
   If(CountIf(colMenuItems,blnCompleted) > 0, 
 
      // If there are items completed, Count them then
      // divide that by the count of items that have minutes
      (CountIf(colMenuItems,blnCompleted) /
      (CountIf(colMenuItems,intTime > 0)) * 100),
      
      // No items completed, means 0
      0
   ),
 
   // Round up to 0 decimal places
   0 
 
) & "% completed"
    Align Align.Left

  15. Now click on lblAppTitle in the Tree view
  16. Update its properties as follows

    Property Value
    X (App.Width - Self.Width) / 2
    Width 450
    Text gblAppTitle
    Align Align.Center

  17. Last up, click on lblMinsRemaining
  18. Update its properties as follows

    Property Value
    X App.Width - Self.Width - 20
    Text Sum(colMenuItems, intTime) - Sum(Filter(colMenuItems, blnCompleted), intTime) & " minutes remaining"
    Align Align.Right

  19. Our screen header is done
