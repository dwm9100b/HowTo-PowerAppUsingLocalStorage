We will be using a gallery control to create our lesson menu.

  1. We will be working with the conLessonMenu container
  2. So make sure the container is selected in the Tree view
  3. Then from the top toolbar, click on +Insert
  4. Search for and select the Blank Vertical gallery control
  5. When prompted for data source, select the colMenuItems
  6. In the right properties pane, under Layout, select Title
  7. In the property above Layout, click on Edit of Fields
  8. I suspect the Title has already defaulted to strTitle, if not do so
  9. Hmmmm you say, I don't see anything
  10. In the left Tree view, right click on the App, click on Run OnStart
  11. You should now see data in your gallery, let's go clean it up
  12. Again in Tree view, rename your gallery to: galLessonMenu
  13. In Tree view expand the gallery control, galLessonMenu
  14. Rename Title: lbLessonlTitle
  15. Delete the Next Arrow icon
  16. Rename the Separator: rectLessonSeparator
  17. Move lblLessonSeparator to the bottom, via the â€¦ menu, reorder, Send to back
  18. Click on lblLessonTitle under the galLessonMenu control in the Tree view
  19. From the top toolbar, click on +Insert
  20. Expand the Icons section, scroll and select the check mark icon
  21. Rename the icon: icoLessonCompleted
  22. Don't worry yet about properties for any of these controls, we'll get to them
  23. One more control, click on the icoLessonCompleted control in galLessonMenu
  24. From the top toolbar, click on +Insert
  25. Select Text label
  26. Rename the Text label: lblLessonTime
  27. Ok let's go update some of the properties for each control
  28. First up, properties for galLessonMenu, click on the galLessonMenu control

    Property Value
    X 0
    Y 0
    Width Parent.Width - 10
    TemplateSize 50
    TemplateFill If(ThisItem.IsSelected, Color.LightGray, Color.White)
    Height Parent.Height
    AlignInContainer AlignInContainer.Start

  29. Now properties for icoLessonCompleted, click on the icoLessonCompleted control

    Property Value
    X 5
    Y 0
    Width Hard coded at 25
    Visible ThisItem.blnCompleted
    Height Parent.TemplateHeight
    Color Color.Green

  30. Let's update the properties for lblLessonTime, click on the lblLessonTime control

    Property Value
    X Parent.TemplateWidth - Self.Width - 3
    Y 0
    Wrap false
    Width 53
    Visible ThisItem.intTime > 0
    VerticalAlign VerticalAlign.Middle
    Text ThisItem.intTime & " min"
    Size 8
    Height Parent.TemplateHeight
    Align Align.Right

  31. Next let's update properties for lblLessonTitle, click on the lblLessonTitle control

    Property Value
    X icoLessonCompleted.X + icoLessonCompleted.Width + 5
    Y 0
    Wrap false
    Width lblLessonTime.X - Self.X
    VerticalAlign VerticalAlign.Middle
    Text ThisItem.strTitle
    Size 11
    Height Parent.TemplateHeight

  32. Final step on this gallery section is to add a divider between the gallery and the content
  33. Make sure you have conLessonMenu selected in the Tree view
  34. Then from the top toolbar, click on +Insert
  35. Locate and select Rectangle, under Shapes section
  36. Rename this control: rectVerticalDivider
  37. Update the properties for rectVerticalDivider, click on the rectVerticalDivider control

    Property Value
    X 0
    Y 0
    Width 2
    Height Parent.Height
    Fill RGBA(56, 96, 178, 1)

  38. One last item, click on the App object in Tree view
  39. Select the OnStart property
  40. At the very bottom of the OnStart code, add the following

    // Make sure our gallery has been reset
    Reset(galLessonMenu);

  41. Good time to not only save the app, but go ahead and do a publish
