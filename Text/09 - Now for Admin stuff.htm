Sometimes an app needs a way to work with global Admin type of actions, we do. We'll call this Admin stuff.

Here we will provide a means to clear all the Completed flags along with another option to actually delete the entire data file stored locally. The data file will be recreated when the app is run again or refreshed.

  1. We will do this as bulk as we can
  2. Over in the Tree view on the left
  3. Click on conAdminStuff
  4. We will have you click on +Insert from the top toolbar for each of the following, if you refrain from clicking on any controls, and just do the insert, they will be inserted in correct order

    Create Control Rename to
    Text label lblAdminDescription
    Text label lblResetDescription
    Button btnAdminReset
    Text label lblDeleteDescription
    Button btnAdminDelete

  5. Once you have added all 5 controls lets go change some properties
  6. First, select the 3 Text label controls together and change the following group properties

    Property Value
    AutoHeight true
    Size 13
    Width 450
    Wrap true

  7. Then select the 2 button controls together and edit the following group properties

    Property Value
    Height 40
    Size 15
    Width 200

  8. Click on lblAdminDescription only and add this to the Text property

    Property Value
    Text "This page will be used for any Admin related functions"

  9. Now click on lblResetDescription and add this to the Text property

    Property Value
    Text "First up will be our reset completed function. As the name implies pressing this button will reset all completed to be false. It will also save this data which will let you start over."

  10. Click on lblDeleteDescription and add this to the Text property

    Property Value
    Text "Next is the delete data file button. In case there is an update to the data, app or the app just not operating right. Click this button. After pressing the Delete Data File button, close and reload the app (or just refresh in the web view). This allows for a new data file to be created, clean slate."

  11. If you want to see how things are progressing visually, hold the Alt key down and
    press Admin stuff in the left gallery menu of the app. There ya go!

  12. Great! Now let's work with our buttons, click on btnAdminReset, update its properties

    Property Value
    Text "Reset Completed"
    OnSelect 
    // Reset all completed bits to false
 UpdateIf(colMenuItems, 
   blnCompleted, 
   {blnCompleted: false});
 
// Now do a reset on the gallery
Reset(galLessonMenu);
 
// Make sure to save the data status
SaveData(colMenuItems, gblDataFile);

  13. Click on btnAdminDelete and update its properties as follows

    Property Value
    Text "Delete Data File"
    OnSelect ClearData(gblDataFile)

  14. One last item to update, click on the conLessonContent control in the Tree view
  15. Update the following property

    Property Value
    Visible !conAdminStuff.Visible

  16. This completes the Admin section for now. If everything went in correctly you should be able to preview the app and only see the Admin section when its item in the left gallery menu is clicked on. Of course you would have to have set the intId for Admin to be 1000.
