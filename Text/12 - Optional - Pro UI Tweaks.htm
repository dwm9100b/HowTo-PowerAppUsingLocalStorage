I highly recommend that you follow along and do this exercise, even though it is optional.

First let's show you what we are looking to fix (clean up) in the user interface (UI) of almost all Power Apps.

If you have been following along and completed up until now, go ahead and run your app. Once its running, hover your mouse over any of the text in our top screen header. Did you notice that the cursor changed to a text insert icon? Now do the same over the text in the page header. Also go ahead and hover your mouse over any of the items in the left menu gallery. Notice also that when the cursor goes over any text, it changes to that insert text icon.

I'll show you how to fix this and make your app look a bit more polished.

  1. First one is easy, lets work on the top screen header
  2. Expand the conScreenHeader container control in the Tree view
  3. Then click on the lblPctComplete control, or whichever control is first
  4. Go up to the top tool bar and click on +Insert
  5. Locate and select the Rectangle shape
  6. Rename it: rectScreenHeaderCursor
  7. It should by default be the first control in the container
  8. If it isn't, make it so
  9. Now update the properties as follows for rectScreenHeaderCursor

    Property Value
    X Parent.X
    Y Parent.Y
    Width Parent.Width
    Height Parent.Height
    Fill Color.Transparent

  10. There, now the mouse cursor will no longer show that silly insert icon
  11. Next up, lets work on the left menu
  12. Expand the conLessonMenu container in the Tree view
  13. Expand the galLessonMenu control also
  14. Click on the lblLessonTime control, or whichever control is first
  15. From the top toolbar, +Insert
  16. Expand the Icons section, click on the Add icon
  17. Actually any icon will do
  18. Rename to: icoLessonCursor
  19. This should put the icon above the lblLessonTime control
  20. If not make this so, it should be the first control in the gallery
  21. Now let's update the properties of the control, icoLessonCursor

    Property Value
    X 0
    Y 0
    Width Parent.TemplateWidth
    Height Parent.TemplateHeight
    Color Color.Transparent

  22. Awesome! Now if you test it, you'll get a nice hand icon when over the menu
  23. Ok final update, also fairly easy
  24. Expand the conLessonHeader control in the Tree view
  25. Click on the lblPageTitle control, or whichever control is first
  26. From the top toolbar, click on +Insert
  27. Locate and select the Rectangle shape
  28. Rename it: rectPageTitleCursor
  29. Make sure this is the first control in the container
  30. Let's update the properties for the rectPageTitleCursor control

    Property Value
    X lblPageTitle.X
    Y lblPageTitle.Y
    Width lblPageTitle.Width
    Height lblPageTitle.Height
    Fill Color.Transparent

  31. All done! Save, publish and run your app.
  32. Congratulations! Your app now has that polished professional look and feel!
