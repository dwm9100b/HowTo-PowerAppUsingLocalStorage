Let's now create the header for the lesson content

  1. Click on to select conLessonHeader in the Tree view
  2. Then from the top toolbar, click on +Insert
  3. Locate and select the Text label control
  4. Rename this one: lblPageTitle
  5. Now click on +Insert again
  6. Locate and select the Button control
  7. Rename this: btnLessonCompleted
  8. Now select the lblPageTitle control
  9. Update its properties as follows

    Property Value
    X 15
    Y 0
    Width Parent.Width - btnLessonCompleted.Width - 50
    VerticalAlign VerticalAlign.Middle
    Text galLessonMenu.Selected.strTitle
    Size 20
    Height Parent.Height

  10. Now let's have you update the button properties
  11. In the Tree view, click on to select btnLessonCompleted
  12. Update its properties as follows

    Property Value
    X Parent.Width - Self.Width - 15
    Y (Parent.Height - Self.Height) / 2
    Visible galLessonMenu.Selected.intTime > 0
    VerticalAlign VerticalAlign.Middle
    Text If(galLessonMenu.Selected.blnCompleted, "Reset Completed", "Completed")
    Size 12
    OnSelect 
    // Get the selected record
Set(varTempSelected,
   LookUp(colMenuItems, galLessonMenu.Selected.intId=intId));
 
// Toggle the completed bit, false to true/true to false
Patch(colMenuItems, varTempSelected,
   {blnCompleted: !varTempSelected.blnCompleted});
 
// Make sure to save the data status
SaveData(colMenuItems, gblDataFile);
    Height 28
    Align Align.Center

  13. Great! All done with the page header... save, next
